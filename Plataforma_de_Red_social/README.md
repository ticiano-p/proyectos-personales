# Red Social - Proyecto Vue.js + Supabase

Este proyecto es un **sitio web tipo red social** desarrollado con **Vue.js** para el frontend y **Supabase** como backend y base de datos.  

El sitio permite a los usuarios interactuar de manera completa con la plataforma y entre ellos, ofreciendo funciones típicas de una red social moderna.

---

## Funcionalidades

1. **Publicaciones**
   - Subir publicaciones con **foto y texto**.
   - **Editar y eliminar** tus propias publicaciones.
   - Visualizar publicaciones de otros usuarios.

2. **Comentarios**
   - Comentar las publicaciones de otros.
   - **Eliminar tus comentarios** si lo deseas.

3. **Perfiles de usuario**
   - Ver el **perfil de otros usuarios** con sus publicaciones.
   - **Editar tu información de usuario**.
   - Colocar una **foto de perfil**.

4. **Mensajería privada**
   - Enviar y recibir **mensajes privados** entre usuarios.

5. **Gestión de contenido**
   - Control total sobre tus publicaciones y comentarios.
   - Funcionalidades de edición y eliminación para mantener tu perfil actualizado.

---

## Tecnologías utilizadas

- **Frontend:** Vue.js, JavaScript, HTML, tailwind CSS  
- **Backend / Base de datos:** Supabase (PostgreSQL, Auth, Storage)  
- **Almacenamiento de imágenes:** Supabase Storage  
- **Autenticación:** Supabase Auth (usuarios registrados y login seguro)

---

## Estructura del proyecto

- `src/` → código fuente de la aplicación Vue.js  
- `components/` → componentes reutilizables del frontend  
- `pages/` → vistas principales de la app (perfil, publicaciones, mensajes, etc.)  
- `services/` → configuración de la base de datos y storage (backup.sql opcional)
